Agendamento de TCC

Este projeto foi concebido por meio da utilização da Linguagem de Modelagem Unificada (UML) em um contexto profissional, com o propósito de implementar uma interface gráfica utilizando a linguagem de programação Java.

Durante a execução deste trabalho, empregou-se o Sistema de Gerenciamento de Banco de Dados MySQL Workbench para a criação das tabelas necessárias. Adicionalmente, elaborou-se um diagrama de comportamento para tais tabelas, e uma visão (view) foi criada para apresentar uma representação de uma "agenda".

A totalidade do código reside no ambiente do banco de dados, oferecendo funcionalidades de CRUD (Create, Read, Update, Delete), possibilitando a criação, leitura, edição e exclusão de dados. Adicionalmente, foram implementados tratamentos de exceções para a maioria das operações, conferindo robustez ao sistema e tornando-o apto para uso local.

Projeto de Histórico de Salas

O Projeto de Histórico de Salas consiste na geração de códigos QR exclusivos para cada sala, com o propósito de afixar esses códigos nas portas, proporcionando acesso imediato aos usuários para visualizarem informações pertinentes a cada sala específica.

A concepção e execução desse projeto foram realizadas de forma colaborativa em um ambiente de trabalho em grupo. As responsabilidades foram distribuídas entre as equipes de desenvolvimento de backend e frontend. Utilizamos as tecnologias CodeIgniter e PHP para o backend, enquanto o banco de dados MySQL foi administrado por meio do PhpMyAdmin. Adicionalmente, empregamos a abordagem de transferência de arquivos via FTP e SSH para a implementação do projeto em nosso ambiente de hospedagem.

O sistema conta com funcionalidades cruciais para a administração, permitindo que o administrador crie e exclua salas sem a necessidade de acesso direto ao banco de dados, simplificando assim as operações essenciais para a gestão eficaz do sistema.
